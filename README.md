# FretFlowDjango
Backend for Fret Flow

# Set up
Activate the python  virtual environment:
```
  .\env\Scripts\activate
```
To deactive the virtual environment:
```
  deactivate
```
Migrate changes:
```
  python manage.py migrate
```
Run the server:
```
  python manage.py runserver
```
Server runs on:
  http://127.0.0.1:8000/
